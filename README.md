# Inventory Manager (Система складського обліку)

Програмний комплекс для автоматизації процесів обліку інвентарю на складі. Проєкт розроблено в межах курсової роботи.

## 🚀 Основний функціонал
* **Управління товарами:** Підтримка різних зон зберігання (COLD, DRY, SECURE).
* **Поліморфна ієрархія:** Розрахунок вартості зберігання залежно від категорії товару (Швидкопсувні, Електроніка).
* **Персистентність даних:** Повне збереження та відновлення стану системи (pickle), експорт у CSV та читання конфігурації з JSON.
* **CLI-інтерфейс:** Зручне текстове меню із захистом від введення некоректних даних (Fault Tolerance).
